## Debian Packages: Basic

The following basic packages are always required
(account: <span style="color:red; font-weight:bold; font-size:larger;">root</span>).

```
{% raw %}
DEBS="
aptitude
vim
"
apt update && apt upgrade -y && apt install -y $DEBS
{% endraw %}
```

## Debian Packages: Compiling Kernel

You need these packages for compiling the Linux Kernel
(account: <span style="color:red; font-weight:bold; font-size:larger;">root</span>).

```
{% raw %}
DEBS="
bc
bison
build-essential
flex
git
gnupg
libelf-dev
libncurses-dev
libssl-dev
rsync
"
apt update && apt upgrade -y && apt install -y $DEBS
{% endraw %}
```
